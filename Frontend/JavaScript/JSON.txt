🧩 What Is JSON?

JSON stands for JavaScript Object Notation.

It's a lightweight data format used to store and exchange data.
It's easy for humans to read and machines to parse.

🧩 JSON vs JavaScript Object

They look similar, but JSON is a string format, 
while a JavaScript object is an actual object.

🧩 JSON (as a string):

{
  "name": "Alice",
  "age": 25, 
  "isStudent": true
}
Keys and strings must be in double quotes
No functions, comments, or undefined values

🧩 JavaScript Object:

{
  name: "Alice",
  age: 25,
  isStudent: true 
}

🧩 Converting JSON and JavaScript Object

👉 JavaScript Object → JSON String

const person = {
  name: "Alice",
  age: 25,
  isStudent: true
};

const jsonString = JSON.stringify(person);
console.log(jsonString);
// Output: '{"name":"Alice","age":25,"isStudent":true}'

👉 JSON String → JavaScript Object

const jsonString = '{"name":"Alice","age":25,"isStudent":true}';

const person = JSON.parse(jsonString);
console.log(person.name); // Output: Alice
 
🧩 Things to Remember

|      Valid in JSON        |    Invalid in JSON    |
| ------------------------- | --------------------- |
| Numbers                   | undefined             |
| Strings (double quotes)   | Functions             |
| Booleans ( true / false ) | Comments              |
| null                      | Single quotes in keys |

🧩 Fetching Data from an API
Application Programming Interface 

fetch('https://api.example.com/user')
  .then(response => response.json())
  .then(data => {
    console.log(data); // This is now a JS object
  });
