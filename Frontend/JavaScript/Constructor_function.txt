
ğŸª What is a Constructor Function?

A constructor function is a special kind of function 
used to create objects quickly.

    Instead of writing objects one by one, 
    we use constructor functions.

    constructor function names start with Capital Letter.

Example without constructor

    // Creating multiple objects manually 
    let student1 = { name: "Arun", age: 20 };
    let student2 = { name: "Meera", age: 22 };

Example with constructor

    function Student(name, age) {
        this.name = name;  
        this.age = age;
    }

    // Creating objects using constructor
    let s1 = new Student("Arun", 20);
    let s2 = new Student("Meera", 22);

    console.log(s1.name); // Arun
    console.log(s2.age);  // 22
