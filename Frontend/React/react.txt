React is a JavaScript library (not a full framework) 
created by Facebook (Meta).

It helps developers build user interfaces (UIs), 
especially for web apps.

Its main power is handling dynamic, interactive, 
and reusable components.

Instead of reloading the whole page, 
React updates only the parts of the UI 
that need to change (using something called the Virtual DOM).
