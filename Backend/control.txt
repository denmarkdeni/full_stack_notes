Jumping Statements

👉 These statements interrupt the normal flow of execution.

1. break → Stop the loop completely

for i in range(5):
    if i == 3:
        break
    print(i)

Output: 0 1 2

👉 Loop stops immediately when condition is met

2. continue → Skip current iteration

for i in range(5):
    if i == 2:
        continue
    print(i)

Output: 0 1 3 4

👉 It skips only that step, not the loop

👉 Break and Continue used only inside loops 

pass → Do nothing (placeholder)

👉 Used anywhere (loop, function, class)

for i in range(5):
    if i == 2:
        pass
    print(i)

👉 No effect — just avoids syntax error

4. return → Exit function + send value

👉 Used only inside functions

def test():
    print("Start")
    return
    print("End")

test()

Output: Start

👉 Function stops at return

Return with value

def add(a, b):
    return a + b

| Keyword  | Where used | Effect                  |
| -------- | ---------- | ----------------------- |
| break    | loop       | stops loop completely   |
| continue | loop       | skips current iteration |
| pass     | anywhere   | does nothing            |
| return   | function   | exits function          |
