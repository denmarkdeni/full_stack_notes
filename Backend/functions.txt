Functions

    👉 A function is a reusable block of code.

    def greet():
        print("Hello")

    greet()

    If you’re not using functions:

    👉 You’re repeating code
    👉 Your code becomes unmaintainable

Parameters / Arguments

    def greet(name):
        print("Hello", name)

    greet("Denny")

Return vs Print

    def add(a, b):
        return a + b

    result = add(2, 3)
    print(result)

    return → gives value back
    print → just displays

Default Parameters

    def greet(name="User"):
        print("Hello", name)

    greet()        # Hello User
    greet("Deni")  # Hello Deni

Keyword Arguments

    def student(name, age):
        print(name, age)

    student(age=21, name="Deni")

*args (multiple values)

    def total(*numbers):
        return sum(numbers)

    print(total(1, 2, 3, 4))

**kwargs (key-value)

    def details(**data):
        print(data)

    details(name="Deni", age=21)

Lambda (one-line function)

    square = lambda x: x * x
    print(square(5))

Function Scope

    x = 10

    def test():
        x = 5
        print(x)

    test()      # 5
    print(x)    # 10

    👉 Local ≠ Global

Recursion ( Function calling itself )

    def fact(n):
        if n == 1:
            return 1
        return n * fact(n-1)

First-Class Functions

    def greet():
        return "Hi"

    x = greet
    print(x())

Nested Functions

    def outer():
        def inner():
            print("Inside")
        inner()

Decorators

    def my_decorator(func):
        def wrapper():
            print("Before")
            func()
            print("After")
        return wrapper
