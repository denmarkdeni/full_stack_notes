MVT Architecture

Models - Views - Templates

👉 Model - database python classes

👉 Views - Logic layer (request and response)

👉 Templates - Presentation Layer

ORM - Object Relational Mapping

👉 Helps Python to communicate with the DB

Migration 

python manage.py makemigrations
    👉 saves model changes
python manage.py migrate
    👉 migrates the changes to DB

Project - main center , Configuration 
App - Particular Feature
