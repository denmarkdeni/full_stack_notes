Django QuerySet API   

	all():              Grabs every object.
	filter(kwargs):     Filters matches.
	get(kwargs):        Gets one exact match.
	create(kwargs):     Creates/saves new object.
	update(kwargs):     Updates matches.
	delete():           Deletes matches.
	count():            Counts objects.
	exists():           Checks existence.
	order_by(*fields):  Sorts by fields ascending/descending.
	exclude(kwargs):    Excludes matches.
