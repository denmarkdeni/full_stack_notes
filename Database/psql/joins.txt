ðŸ”‘ INNER JOIN

Returns only rows with matching values in both tables.

SELECT s.student_id, s.name, c.class_name
FROM students s
INNER JOIN classes c
ON s.class_id = c.class_id;

student_id | name  | class_name
-----------+-------+-----------
1          | John  | Physics
2          | Alice | Chemistry

ðŸ”‘ LEFT JOIN

Returns all rows from the left table (students),
with NULL if no match in classes.

SELECT s.student_id, s.name, c.class_name
FROM students s
LEFT JOIN classes c
ON s.class_id = c.class_id;

student_id | name  | class_name
-----------+-------+-----------
1          | John  | Physics
2          | Alice | Chemistry
3          | Bob   | NULL
4          | Emma  | NULL

ðŸ”‘ RIGHT JOIN

Returns all rows from the right table (classes), 
with NULL if no match in students.

SELECT s.student_id, s.name, c.class_name
FROM students s
RIGHT JOIN classes c
ON s.class_id = c.class_id;

student_id | name  | class_name
-----------+-------+-----------
1          | John  | Physics
2          | Alice | Chemistry
NULL       | NULL  | Biology

Returns all rows when there is a match in either table.

FROM students s
FULL JOIN classes c
ON s.class_id = c.class_id;

student_id | name  | class_name
-----------+-------+-----------
1          | John  | Physics
2          | Alice | Chemistry
3          | Bob   | NULL
4          | Emma  | NULL
NULL       | NULL  | Biology

ðŸ”‘ SELF JOIN

A table joined with itself.

SELECT a.name AS student1, b.name AS student2, a.class_id
FROM students a
JOIN students b
ON a.class_id = b.class_id
AND a.student_id < b.student_id;

student1 | student2 | class_id
---------+----------+----------
John     | Alice    | 101
